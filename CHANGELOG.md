# CHANGELOG

## [Uncapper v2.4.1] - 2023-10-15
### Added
- **NVIDIA Reflex Support**: Introduced support for NVIDIA Reflex, enhancing responsiveness and reducing input lag for a smoother gameplay experience.
- **Custom Resolution Profiles**: Users can now save and switch between multiple resolution profiles with customizable settings for each profile.
  
### Fixed
- **Minor Bug Fixes**: Resolved various stability issues reported by users leading to improved overall performance.
- **Launcher Crash**: Fixed an issue that caused the application to crash when launching certain games.

### Changed
- **User Interface Update**: Refined the UI to improve navigation and accessibility with clearer labels and enhanced tooltips.

---

## [Uncapper v2.3.0] - 2023-08-02
### Added
- **FPS Limiter Options**: Introduced new options for precise FPS limiting (30, 60, 120, 144 Hz), catering to specific user needs.
- **Game Compatibility List**: A comprehensive list of optimized and tested games is now available, helping users understand which titles benefit most from the uncapper.

### Fixed
- **Screen Flickering**: Fixed a major issue with screen flickering in certain games when using the uncapper.
- **Overlay Conflicts**: Resolved conflicts with common gaming overlays (e.g., Discord, Steam) that caused performance drops.

### Changed
- **Enhanced Compatibility Modes**: Improved compatibility modes for legacy titles, ensuring older games run smoothly without FPS restrictions.

---

## [Uncapper v2.1.0] - 2023-05-10
### Added
- **DirectX 12 Support**: Added support for DirectX 12 games, expanding compatibility and performance enhancement options for modern titles.
- **Performance Logging**: Users can now enable performance logging to monitor FPS changes over time and analyze game performance.

### Fixed
- **Detection of V-Sync**: Improved the detection of V-Sync settings in various games to provide more accurate uncapping results.
- **UI Responsiveness**: Fixed lag issues in the settings menu when rapidly switching between options.

### Changed
- **Updated Documentation**: Provided more detailed guides and troubleshooting steps directly within the app for user reference.
